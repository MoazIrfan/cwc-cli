## Usage

```sh
# Show all World Cup matches schedule.
npx wc-cli

# Show today matches schedule.
npx wc-cli today

# Show schedule for country matches.
npx wc-cli pakistan

# For multiword countries.
npx wc-cli "New Zealand"
npx wc-cli "new zealand"
```
