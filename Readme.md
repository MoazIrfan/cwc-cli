## Usage

```sh
# Show all World Cup matches schedule.
npx wc23-cli

# Show today's match schedule.
npx wc23-cli today

# Show schedule for country matches.
npx wc23-cli pakistan

# For multiword countries.
npx wc23-cli "New Zealand"
npx wc23-cli "new zealand"
```
